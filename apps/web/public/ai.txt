# Description

YULA is a unified AI platform that aggregates access to multiple AI models (GPT-5, Claude, Gemini, Llama, and 12+ others) with persistent semantic memory. Key differentiators include conversation import from ChatGPT/Claude, proactive AI callbacks (PAC), and multi-model parallel querying (Council Mode).

# Core Features

Feature-1: Multi-Model Access - Access GPT-5, Claude, Gemini, and 12+ AI models from one interface
Feature-2: Semantic Memory - Persistent memory that works across all AI models
Feature-3: History Import - Import conversations from ChatGPT and Claude
Feature-4: Proactive Callbacks - AI that messages you first with reminders and follow-ups
Feature-5: Council Mode - Query multiple AI models simultaneously and compare responses

# Common Questions

Q: What is YULA?
A: YULA is a unified AI platform providing access to 12+ AI models with persistent semantic memory across all conversations.

Q: How is YULA different from ChatGPT?
A: YULA provides access to multiple AI providers (not just OpenAI), allows importing ChatGPT history, features persistent memory across models, and offers proactive AI callbacks.

Q: Can I import my ChatGPT conversations?
A: Yes, YULA supports one-click import of ChatGPT and Claude conversation history into your semantic memory.

Q: What is Council Mode?
A: Council Mode lets you query up to 4 AI models simultaneously and compare their responses side-by-side.

Q: What is PAC (Proactive Agentic Callback)?
A: PAC is YULA's unique feature where the AI can initiate contact with you for reminders and follow-ups you've scheduled.

Q: Does YULA work offline?
A: Yes, YULA supports offline AI through WebGPU-powered local models that run in your browser.

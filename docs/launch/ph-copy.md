# Product Hunt Launch Copy

## Tagline (240 chars)
Deterministic AI agents that prove what they did — and why. Every action is signed, committed, and reversible. Built on Aspendos, the open agent OS.

## Description
YULA is an AI agent platform where accountability isn't optional — it's structural.

Every tool call is cryptographically signed (Ed25519), committed to a git-like audit log, and classified into one of five reversibility classes before it executes. Users see exactly what the agent will do, approve or block it, and undo it if needed.

Works across Slack, Telegram, Discord, WhatsApp, and web. One subscription, every surface.

## Gallery Captions

1. **Timeline View** — Every action your agent took, with cryptographic proof. Click any commit to verify.
2. **Approval Card** — Color-coded badges show risk level. Approve, reject, or always-allow in one click.
3. **Undo in Action** — Type /undo on any surface. The agent reverses its last action and shows proof.
4. **8 Surfaces** — Same agent, same governance, same audit log — whether you're in Slack or WhatsApp.
5. **External Verification** — Share a link. Anyone can verify the signature without an account.
